~ Data types :

# data types 
- 1. pre-defined / builtin : int , float , char , double , void , wide character
- 2. user-defined          : class , structure , union , enum , typedef
- 3. derived               : array , function , pointer , reference

# pre-defined / builtin data types 

- int :
    - all the integers are under int data type
    - size 2 or 4 byte
    - eg. 1 , 2 , 3 , 56  etc

- float : 
    - numbers with fewer decimal comes under floating point or float data type
    - size 4 byte
    - In general, 7 decimal digits precision
    - eg. 1.2f , 3.7f , 9e-5f etc

- double :
    - numbers with more decimal comes under double floating poitn or double data type
    - size 8 byte 
    - In general, 15 decimal digits precision
    - eg. 1.2 , 4.6789 , 3e-10 etc
 
- char :  
    - every char (alphabet , numbers , special characters etc ..) comes under char 
    - size 1 byte
    - eg. 'f' , 'l' , 'j' .. etc

- wide character : 
    - is also a character data type but this data type has a size greater than the normal
    - size 8 byte
    - represented as wchar_t
    - it store the ASCII value of that character

- void :
    - it defines absence of value
